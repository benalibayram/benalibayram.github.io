Circos: Fonksiyonel bağlantısallık

Eylül 2018

Circos kurulumu:

Kurulum için Circos sitesini takip ettim.

Öncelikle Strawberry Perl kurulumunu yaptım. Perl kurulduktan sonra circos-0.69-5.tgz'yi indirip açtım. Bendeki konumu D:\StrawberryCircos\.

Circos'un eksik modülleri için perl D:\StrawberryCircos\circos-0.69-5\bin\circos -modules komutunu çalıştırdım. Eksik modüller listelendi. Yüklemel için ise D:\Strawberry\perl\bin\cpan çalıştırdım. Açılan cmd ekranından sırasıyla eksik modülleri aşağıdaki gibi yükledim. 

cpan> install Config::General
							...
							cpan> install Text::Format

Example klasöründe PowerShell çalıştırıp D:\StrawberryCircos\circos-0.69-5\example> perl D:\StrawberryCircos\circos-0.69-5\bin\circos -conf etc\circos.conf yazınca örneği üretti. Yani kurulum tamam.

Circos'ta Connectogram Hazırlamak:

Circos'ta genetik terimler kullanıldığı için beyin bağlantısallığındaki karşılıkları: kromozom'un beyin lobları, band'ın beyin bölgeleri olacaktır. Hazırladığımız data dosyasına karyotype denilecektir.

Uygun format = chr – ID Label(grafikte görünmesini isteğiniz isim) başlangıç bitiş(kromozomun boyutunu belirlemek için) renk

Örnek chr - fro_l FRONTAL 0 212302 white frontal lobda 212302 vokselimin olduğunu ve grafikte bu bölgenin FRONTAL olarak görünmesini istediğimi belirtmiş oluyorum. Renklerle alakalı yardım için de etc/color.conf belgesine bakabilirsiniz. Lobların altındaki beyin bölgelerini belirtmek için de uygun format; band kromozom adı ID Label başlangıç bitiş renk.
